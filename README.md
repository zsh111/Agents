# agents

该仓库由 Copilot 初始化为一个最小的 Python 项目骨架。

快速开始：

1. 创建虚拟环境并安装依赖：

```zsh
make setup
```

2. 运行测试：

```zsh
make test
```

项目布局：

- `pyproject.toml`：项目元数据。
- `requirements.txt`：运行时依赖（当前为空或仅包含开发依赖）。
- `src/agents`：包代码。
- `tests/`：pytest 测试。
- `scripts/setup.sh`：创建虚拟环境并安装依赖。
